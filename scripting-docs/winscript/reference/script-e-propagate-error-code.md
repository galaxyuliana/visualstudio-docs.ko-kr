# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE 오류 코드
스크립트 오류를 다른 스레드에서 될 수 있는 호출자에 게 전파 됩니다. 호스트는 호출자에 게 오류 코드를 전달 해야 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
