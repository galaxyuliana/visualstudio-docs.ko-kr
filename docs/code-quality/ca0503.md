# <a name="ca0503"></a>CA0503
속성을 사용 하는 사용 되지 않습니다. 대체 속성 사용

 경고 CA0503 사용 되지 않는 코드 분석 속성 빌드 또는 프로젝트 구성 파일에 값을 포함 하는 경우에 생성 됩니다. 이 경고를 해결 하려면 대체 하는 지정된 된 속성을 사용 합니다.
