# <a name="ca0503"></a>CA0503
속성은 사용 되지 않습니다. 대체 하는 속성 사용

경고 CA0503는 사용 되지 않는 코드 분석 속성에 빌드 또는 프로젝트 구성 파일의 값이 포함 되어 있을 때 생성 됩니다. 이 경고를 해결 하려면 해당 속성을 대체 하는 지정 된 속성을 사용 합니다.
