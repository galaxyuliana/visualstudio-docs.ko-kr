# <a name="ca0501"></a>CA0501
출력 보고서를 읽을 수 없습니다.

 다음 이유 중 하나로 인해 보고서 출력을 쓸 수 없는 경우이 발생 합니다.

- 디스크가 꽉 찼습니다.

- 디렉터리에 쓸 수 없는 권한

- 디렉터리가 존재 하지 않습니다.
