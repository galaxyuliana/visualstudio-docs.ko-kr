# <a name="c28022"></a>C28022
경고 C28022: 함수 클래스 \<classlist1 >이 함수의 함수 클래스를 일치 하지 않는 \<classlist2 > 정의 하는 데 사용 된 typedef의 합니다.

 이 경고는 주석에 오류가 있는 경우 보고됩니다. Typedef와 함수 자체에 `_Function_class_` 주석을 있지만 일치 하지 않습니다. 둘 다 사용 하는 경우 일치 해야 합니다.
