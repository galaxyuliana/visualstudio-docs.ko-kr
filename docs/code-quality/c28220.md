# <a name="c28220"></a>C28220
경고 C28220: 주석에 있는 매개 변수에 정수 식이 필요합니다.

 이 경고는 주석의 매개 변수는 정수 식 이어야 하는데 및 일부 다른 형식이 발견 되었음을 나타냅니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
