# <a name="tree-menu-commands"></a>트리 메뉴 명령
합니다 **트리** 메뉴 명령을 확장 또는 축소 뷰를 제공 합니다.

 **한 수준 확장** 한 수준 아래에 표시 하 여 시스템 뷰에서 현재 선택한 항목을 확장 합니다.

 **분기 확장** 현재 선택된 된 항목 아래의 모든 수준을 확장 합니다.

 **모두 확장** 확장 및 모든 수준에서 시스템 뷰를 표시 합니다.

 **축소** 현재 선택된 된 항목 아래에 있는 모든 수준을 축소 합니다.
