# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 6997: 이 위치의 주석은 의미가 없으며 무시 됩니다.  
  
 `extern "C" {...}`에 주석을 적용할 수 없습니다. 특정 개체에 주석을 적용하십시오.
