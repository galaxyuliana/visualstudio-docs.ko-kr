# <a name="mparent-field"></a>m_parent 필드

작업의 부모에 대 한 지원 필드 <xref:System.Threading.Tasks.Task>합니다.

**네임스페이스:** <xref:System.Threading.Tasks?displayProperty=fullName>

**어셈블리:** mscorlib (에서 *mscorlib.dll*)
